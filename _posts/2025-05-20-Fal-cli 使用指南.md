## 安装

```bash
pip install fal
```

## 登录

```bash
fal auth login
```

默认会自动跳转页面，如果没有跳转或使用的Linux系统，需要手动复制链接在浏览器打开后登录

登录后还需要输入数字选择使用的账户，默认输入1即可

## 生成密钥

```bash
fal keys create --scope API
```
